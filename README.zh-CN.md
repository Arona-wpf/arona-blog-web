# arona-blog-web

🌍 [English](./README.md) | 中文

这是Aronaの小屋 网站的前端web服务

功能还在逐步开发中……

## 项目架构

框架：Vue 3

语言：TypeScript 5.9.3

构建工具：Vite 8

状态管理：Pinia 3

路由：Vue Router 5

国际化：Vue I18n 11

样式：Tailwind CSS 4

UI组件库：shadcn-vue (reka-ui)

## 项目部署

```sh
  yarn install
```

### 编译和开发热重载

```sh
  yarn dev
```

### 类型检查、编译和生产环境打包

```sh
  yarn build
```

### 类型检查

```sh
  yarn type-check
```

### 代码规范检查

```sh
  yarn lint
```
